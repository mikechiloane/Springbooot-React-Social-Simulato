## Technologies Used
 - Springboot
 - Kafka - Confluent Clound
 - Okta
 - Spring Cloud

## Features

- Location tracking and updating
- Real-time Messaging 
- Secure Login and simple signup process
- Map direction service
- Visit anyone registered on the platform

## Services
 - Auth 
 - Gateway
 - Location
 - User
 - Message
